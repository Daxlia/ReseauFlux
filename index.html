Abstract

ML-guided metabolic intervention ranking

RéseauFlux is a machine learning pipeline designed to rank metabolic interventions across genome-scale metabolic models without requiring exhaustive flux balance analysis for every candidate reaction. The framework combines feature extraction, graph topology, flux variability analysis, and gradient boosting regression to predict high-impact interventions for aerobic succinate overproduction in E. coli. The project explores whether machine learning models can generalize intervention ranking across unseen reactions while preserving biologically meaningful metabolic behavior. The workflow integrates COBRApy-based simulations, robustness analysis, pathway-level interpretation, Monte Carlo perturbation experiments, and multi-objective growth-production tradeoff evaluation.

The current implementation focuses on genome-scale E. coli metabolism using the iJO1366 reconstruction and evaluates candidate knockouts through ML-assisted prioritization followed by targeted FBA validation. The objective is not to replace mechanistic modeling, but to reduce the computational burden of large-scale intervention screening while maintaining biologically interpretable results.
